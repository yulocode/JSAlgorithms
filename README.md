# JavaScript Algorithms and data structures

This repository contains various algorithms and data structures implemented in JavaScript. 
The goal is to practice and strengthen my understanding of fundamental concepts in computer science using JavaScript.
<br>
## Main reference 
The primary resource for the material in this repository is the book <a href="https://eloquentjavascript.net">Eloquent JavaScript.</a>
This book offers a comprehensive guide to JavaScript, from basic syntax to more advanced topics like higher-order functions, asynchronous programming, and data structures.
